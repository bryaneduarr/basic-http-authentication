### Comandos utilizados
```
npm init -y
```
```
npm install typescript ts-node nodemon @types/express @types/node --save-dev
```
```
npx tsc --init
```
```
npm install express dotenv
```
```
npx ts-node src/index.ts
```

```
npm install
```
```
npm run dev
```
